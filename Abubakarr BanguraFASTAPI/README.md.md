# Limkokwing Library Management API

## Project Title
Library API for book search, borrowing, returns, and fine tracking.

## Description
A FastAPI-based REST API that allows library staff and students to:
- Search books by title, author, or category
- Borrow and return books asynchronously
- Track overdue books and calculate fines
- Support concurrent users

## Technologies
- Python 3.9+
- FastAPI
- Uvicorn
- Pydantic

## Installation
```bash
pip install -r requirements.txt
python library_api.py
